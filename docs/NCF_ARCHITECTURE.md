# NCF: Neural Cognitive Framework

## 核心设计原则
1. 无主控架构 - 没有中央控制器，只有器官间的"辅助关系"
2. 事件是唯一通信方式 - 器官间无直接内存访问
3. 智能 = 事件流的自组织
4. 感知即为一切 - 连续的、含噪的感知流是自我涌现的基础

## 认知器官
- 听觉皮层 - 处理声音事件
- 视觉皮层 - 处理视觉事件
- 感觉皮层 - 处理触觉事件
- 顶叶 - 空间感知整合
- 额叶 - 决策与计划
- 布洛卡区 - 语言生成

## 事件类型
- spike(type, data, timestamp) - 神经脉冲事件
- broadcast(source, type, data) - 广播事件
- hormone(m, g) - 内分泌事件（m=浓度, g=梯度）

## 通信规则
- 器官间只能通过事件通信
- 无直接内存访问
- 事件通过Fabric调度

## 意识流
[输入] -> [丘脑注意门控] -> [遗传路径激活] -> [经验路径微调]
-> [本能行为输出] -> [逻辑权限处理] -> [用户权限决策/语言]

## 与EmotionOS的关系
NCF是EmotionOS上层认知架构。
EmotionOS提供基础运行环境，NCF定义认知器官间的协作方式。
