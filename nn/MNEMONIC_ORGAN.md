# 记忆神经网络设计文档 (Mnemonic Organ)

## 概述
三层存储架构的记忆系统，支持联想、遗忘、补全、模糊概念。

## 三层存储
1. 感知缓存 (Sensory Cache) - 200ms滑动窗口，原始高维向量
2. 短期可塑图 STG - 稀疏Hopfield-like网络，Hebbian规则升降边权
3. 长期可塑地形 LTL - 连续吸引子地貌，RNN+混沌动力学

## 联想与补全
- 联想：STG随机游走 -> 高相关子图 -> LTL吸引子采样
- 模糊概念：高斯混合在吸引子盆地边缘保留不确定区域
- 遗忘：情感温度tau（由激素浓度决定）控制噪声冲蚀
- 闪光灯记忆：情绪峰值事件设tau极低

## 自由联想API
Query(q) = LTL.sample(q, temperature = f(H))
- q: 残缺向量/情绪标签/纯激素向量
- f(H): 激素状态 -> 采样温度映射

## 动态突触雕刻
W(t)情感网络的临时权重由STG活跃子图"生长"而来：
1. STG活跃子图 -> 稀疏矩阵
2. 混沌拉伸 -> 直接映射为W(t)
3. 情感核激素向量注入STG作为额外节点标记情绪标签
