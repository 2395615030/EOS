# 脑功能分层建模设计文档

## 三重意识权限模型
| 层级 | 对应功能 | 控制主体 | OS类比 |
|------|----------|----------|--------|
| 用户权限 | 语言、决策、计划 | 主观可控 | User Mode |
| 逻辑权限 | 认知处理、情绪调节 | 半自主 | System Mode |
| 根权限 | 呼吸、心跳、感知I/O | 不可控 | Kernel Mode |

## 模块拆分

### 根权限层
- 丘脑注意力门控 - 多模态输入路由
- 脑干内稳态控制器 - 基础生理维持
- 下丘脑激素调控 - 内分泌调节
- 小脑动作协调 - 反馈校正机制

### 逻辑权限层
- 杏仁核情绪调节
- 基底神经节习惯形成
- 顶叶空间感知整合

### 用户权限层
- 前额叶决策/计划
- 颞叶语言生成与理解
- 枕叶视觉意识层面

## 构建顺序
1. 根权限（打包I/O模块，类似系统引导）
2. 逻辑权限（中间层认知网络）
3. 用户权限（高级意识活动）

## 模块连接
- 先定义标准化接口（输入类型/输出类型/控制信号）
- 使用激活路径选择而非函数调用
- 每层通过预留分层接口连接
