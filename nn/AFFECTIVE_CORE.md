# 情感神经网络设计文档 (Affective Core)

## 概述
非固化权重的情感神经网络，模拟"下丘脑-垂体-肾上腺"轴的硅基实现。
权重不由反向传播训练，而是由记忆网络实时"雕刻"生长。

## 核心方程
H(t+1) = sigmoid(A.H(t) + B.E(t) + C.S(t) + D.M(t) + noise_H)
E(t+1) = tanh(W(t).[H(t);E(t);S(t)] + noise_E)

其中：
- H: 激素向量 (16维)
- E: 情绪向量 (8维)
- S: 外部刺激
- M: 记忆回传
- W(t): 由记忆网络实时雕刻的权重矩阵
- noise: 混沌噪声

## 激素调节
- 阈值触发：激素超标触发全局广播
- 代谢：指数衰减，衰减率受E反馈微调
- 反向影响：高浓度激素改变局部动力学

## 情绪着色
S'(t+1) = S(t+1) concat E(t+1)
刺激被情绪重新编码，形成"情绪-记忆-激素"闭环

## 接口
- input: 刺激向量 S (128-d)
- output: 激素 H (16-d), 情绪 E (8-d)
- read_memory(M): 从记忆网络获取雕刻权重
- write_memory(S,H,E): 写入记忆网络
