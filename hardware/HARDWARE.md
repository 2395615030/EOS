# 物理层硬件设计

## 1. PDMS柔性皮肤

### 材料
| 用途 | 材料 | 型号 |
|------|------|------|
| 基底 | PDMS | Dow Corning Sylgard 184 |
| 软化 | 软胶 | Sylgard 527 / Ecoflex 00-30 |
| 导电 | 液态金属 | EGaIn 或 银纳米线 |
| 表面 | 涂层 | 聚氨酯 / PVA |

### 工艺
- 3D打印模具（树脂光固化）
- 复合配方：184:527 = 1:1（模量50-80kPa）
- 功能层嵌入：半固化时放置蛇形导线
- 接口：ZIF/FPC -> STM32 ADC

### 传感器阵列
- 电阻式/电容式触觉阵列
- FSR402薄膜压力传感器
- 导电PDMS夹绝缘层构成电容触觉点

## 2. 粘菌湿件计算系统

### 四大工位
| 工位 | 状态 | 驻留时长 | 关键 |
|------|------|----------|------|
| 工作位 | 活跃计算 | 48-72h | FPGA实时健康监测 |
| 休眠诱导 | 应激固化 | 24-48h | CaCl2灌注+避光 |
| 休眠库 | 菌核保存 | 数周-数月 | 4度冷藏+紫外灭菌 |
| 复苏位 | 网络再生 | 12-24h | 葡萄糖+红光LED |

### 心脏系统
- 双通道蠕动泵 + 6通旋转阀
- FPGA生成0.1-0.2Hz正弦/方波PWM
- 在线pH/溶解氧/电导率传感器

## 3. NFC PNN边缘节点

### 架构
ADC采样 -> 末梢神经网络(RPS) -> PWM输出(灰质)

### 硬件
- ESP32 / RP2040
- 光敏电阻 + 高精度ADC
- PWM输出引脚
- 可扩展多传感器阵列
